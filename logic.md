# Сценарий взаимодействия с ботом (User Flow)

В данном разделе описана логика работы пользователя и реакции системы.

## Основной цикл работы

| Шаг | Действие пользователя | Реакция бота / Системы |
|:---:|:---|:---|
| **1** | Нажимает кнопку `/start` | Запрашивает выбор языка общения |
| **2** | Вводит/выбирает язык | Производит внутреннюю настройку AI под выбранный язык |
| **3** | — | Задает вопрос: *"В какой стране вы находитесь?"* |
| **4** | Вводит **Страну + Город** | Предлагает выбрать тематику заведений (фильтры) |
| **5** | Выбирает тематику | Формирует комплексный промпт и генерирует ответ через AI |
| **6** | Получает список мест | Выводит результат с кнопками взаимодействия |
| **7** | (Опционально) Нажимает **"Подробная информация"** | Показывает фото, описание и рейтинг. Позволяет поставить свою оценку |
| **8** | — | Переходит в режим ожидания нового цикла |

---

## Технические детали этапов

### 1. Настройка AI
После выбора языка система инициализирует контекст для нейросети, чтобы последующие ответы соответствовали лингвистическим нормам и культурным особенностям региона.

### 2. Формирование промпта
На этапе выбора тематики бот собирает все данные:
* Язык общения
* Геолокация (Страна + Город)
* Интересы пользователя

На основе этих данных создается системный запрос, который отправляется в модель ИИ для получения наиболее релевантных рекомендаций.

### 3. Обратная связь
Функция рейтинга в блоке «Подробная информация» интегрирована с общей системой оценки, что позволяет динамически обновлять рекомендации для других пользователей.
